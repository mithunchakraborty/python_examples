Создание проекта: $ django-admin startproject main .
Подтверждение миграции: $ python3 manage.py migrate
Запуск веб-сервера: $ python3 manage.py runserver
Создание приложения: $ python3 manage.py startapp movies
Создать миграцию: $ python3 manage.py makemigrations
Создать суперпользователя: $ python3 manage.py createsuperuser
Установка ckeditor: $ pip install django-ckeditor
    Далее: $ python3 manage.py collectstatic
